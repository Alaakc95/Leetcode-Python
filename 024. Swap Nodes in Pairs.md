## 24. Swap Nodes in Pairs

https://leetcode.com/problems/swap-nodes-in-pairs/description/

#### Solution:

Time complexity: O(n)ï¼Œ Space complexity: O(1)

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def swapPairs(self, head):
        """
        :type head: ListNode
        :rtype: ListNode
        """
        p1 = s = ListNode(0)
        p1.next = p2 = head

        while(p2 != None and p2.next != None):
            p3 = p2.next
            p1.next = p3
            p2.next = p3.next
            p3.next = p2

            p1, p2 = p2, p2.next

        return s.next
```
